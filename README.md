Overview
Savory By Rei is a fully-featured, single-page recipe management web application built with vanilla HTML, CSS, and JavaScript. It delivers a premium cooking experience with real food photography, 8 stunning visual themes, and robust recipe organization tools — all running entirely in the browser with zero framework dependencies.

Features
🍽️ Real Food Photography
Every recipe features authentic, high-resolution food photography from Unsplash. No placeholders — just mouth-watering photos of actual dishes:

50+ curated food images organized by cuisine type
Smart image matching algorithm pairs recipes with relevant photos
Automatic fallback system ensures images always load
Skeleton loading animations for smooth UX

📋 Smart Organization
15 Recipe Categories — Breakfast through Snacks
Real-time Search across names, ingredients, categories
Smart Sorting by name (A-Z/Z-A), time, or newest
Pagination with smart navigation (shows ellipsis)
Category counts update dynamically
🛒 Shopping List Magic
One-click ingredient collection system:

Add all ingredients from any recipe instantly
Check off items as you shop
Copy entire list to clipboard
Persistent storage (survives browser close)
⚡ Batch Operations
Power user features for managing large collections:

Multi-select recipes via checkbox or click
Batch add to favorites
Bulk add ingredients to shopping list
Batch delete unwanted recipes
Floating action bar appears on selection
💾 Data Portability
Your recipes belong to you:

Export entire collection as JSON backup
Import recipes from JSON files
Bulk Generate up to 3,000 realistic recipes instantly
All data stored locally in localStorage (privacy-first)
